# Core Gendam Mantras & Their Applications

**Purpose**: Opening connection to guru lineage
**Repetitions**: 3x before practice

**Javanese**: Om Sakthi Sakthi Sakthi Om
**Purpose**: Awakening inner power
**Repetitions**: 7x minimum
